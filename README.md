Dotfiles
=========

### How to use

1. cloning repo.
```
cd ~
git clone git@github.com:norangLemon/.dotfiles.git
```

1. make simlink
```
ln -sf ~/.dotfiles/.vimrc ~/.vimrc
ln -sf ~/.dotfiles/.zshrc ~/.zshrc
ln -sf ~/.dotfiles/.bash_profile ~/.bash_profile
ln -sf ~/.dotfiles/.gitconfig ~/.gitconfig
```
